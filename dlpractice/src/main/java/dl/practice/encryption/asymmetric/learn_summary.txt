非对称加密
	将秘钥一分为二公钥和私钥，公钥通过非安全信道进行发放，私钥进行保管
通信模型：
	公布一个公钥，密文使用公钥进行加密，即使在中途被截取，有公钥没有配对的私钥，是不能解密的
特性：
	需要两个秘钥来进行加解密，分别为公钥和私钥
	公钥和私钥相互配对，称为KeyPair
应用：
	公钥加密私钥解密：数据加密
	私钥加密公钥解密：数字签名
优缺点：
	优点：相比对称性密码，安全性更高
	缺点：加解密时间长，速度慢
常用的非对称密码：
	DH秘钥交换算法
	RSA算法--既可以用于数据加密也可以用于数字签名
	ELGamal算法
DH算法
通信模型：
	
RSA算法

通信模型：
	公钥进行加密，私钥进行解密
数学原理:
	加密公式：
		RSA加密公式：密文 = 明文E次方 mod N   公钥：（E,N）
		RSA解密公式：明文 = 密文D次方 mod N   私钥：（D,N）
